# SkillSync

SkillSync is a React-based student dashboard for tracking skills, projects, and internship applications.

## Features

- Skills management
- Projects tracker
- Internship tracker
- Dashboard analytics
- Search and filter functionality
- Edit and update support
- LocalStorage data persistence
- Responsive UI

## Tech Stack

- React
- Vite
- React Router DOM
- CSS

## Pages

- Home
- Dashboard
- Skills
- Projects
- Internships
- Login
